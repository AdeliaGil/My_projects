# Сеть клиник
## О проекте
Проект посвящен подготовке бэклога приложения для сети клиник
## Цель проекта
Совместно с командой составить User Story Map, прототипы и DFD-диаграмму
## Что было сделано
- на основании интервью с с заинтересованными лицами составила список требований, которых коснутся пользовательские истории
- составила User Story Map (в Miro) и выделила на ней MVP, с делальным описанием критериев и сценариев приемки
- дополнила истории прототипами (в Figma):
  1.Раскадровку для просмотра расписания процедурного кабинета
  2. Раскадровку просмотра результатов анализов
  3. Вайрфрейм файла назначения на медикаментозное лечение от врача
- построила контекстную и логическую диаграммы потоков данных (DFD) в draw.io
