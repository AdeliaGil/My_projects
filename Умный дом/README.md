# Умный дом
## О проекте
Проект посвящен разработке требований для первой версии приложения по управлению умным домом
## Цель проекта
Разработать требования и представить заказчикам прототипы экранов, продемонстрировав весь функционал с помощью документа "Программа и методика испытаний"
## Что было сделано
- на основании интервью с заинтересованными лицами создала User Story Map в Miro, выделив на ней MVP и описав из критерии и сценарии приемки
- проанализировала все данные, которые будут использоваться в приложении и создала DFD в Draw.io контекстного и логического уровня
- разработала ER-диаграмму в Draw.io, на которой отобразила все сущности и их атрибуты
- составила "Словарь данных" для единого понимания
- построила интерактивные раскадровки экранов в Figma
- создала документ "Программа и методика испытаний", в соответствии с которым продемонстрировала все функции на интерактивных прототипах
