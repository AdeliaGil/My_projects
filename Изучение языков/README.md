# Изучение языков
## О проекте
Проект посвящен подготовке спецификации требований ПО ко второму релизу приложения для изучения иностранных языков
## Цель проекта
Дополнить спецификацию функциональными требованиями для функции карточек для запоминания новых слов, описать требования к новой функции просмотра видео
## Что было сделано
- составила функциональные требования для функции карточек
- дополнила Use Case, добавив на диаграмму варианты использования функции просмотра видео
- добавила табличные варианты использования для функции просмотра видео
- описала  функциональные требования к видео
- описала атрибуты качества и бизнес-правила для новой функции просмотра видео, а также дополнила глоссарий
