# Интеренет-магазин
## О проекте
Проект посвящен моделированию данных и работе с базой данных интерент-магазина
## Цель проекта
Сформировать логическую модель данных и описать словарь данных
## Что было сделано
- на основании Спецификации требований к ПО, выявила данные приложения
- построила логическую модель данных в нотации Crow’s Foot
- привела полученную модель к третьей нормальной форме (3НФ)
- создала словарь данных, описала в нем все сущности и их атрибуты
- добавила полученную модель и словарь данных в новую версию спецификации требований
