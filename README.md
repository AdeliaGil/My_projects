# My_projects
## 1. Разработка требований и прототипов приложения для управления умным домом.
В работе использовала программы Miro, Draw.io, Figma, Google Docs, и представила заказчикам результат в виде ПМИ.
В папке артефакты: User Story Map, DFD, ER-диаграмма, словарь данных, прототипы экранов пользовательского интерфейса, Программа и методика испытаний.

## 2. Разработка спецификации ПО, для новой версии приложения по изучению иностранных языков.
Дополнила действующую спецификацию требований, обновила UML Use Case и другие разделы.

## 3. Поставка задачи на доработку API.
Построила диаграмму последовательности в draw.io, протестировала существующее API в Swagger, на основании результатов тестирования поставила задачу разработчику.
