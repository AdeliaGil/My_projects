# Онлайн-кинотеатр
## О проекте
Проект посвящен постановке задачи на разработку новой операции API
## Цель проекта
Поставить задачу на доработку API для функции фильтрацию по эксклюзивному контенту сервиса
## Что было сделано
- изучила интервью с представителем онлайн-кинотеатра для понимания его требований
- создала диаграмму последовательности
- протестировала существующее API в Swagger
- поставила задачу на разработку новой операции API
